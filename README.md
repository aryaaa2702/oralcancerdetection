# Oral Cancer Detection using CNN

This project focuses on detecting oral cancer from medical images using a Convolutional Neural Network (CNN). The model is trained to classify images as cancerous or non-cancerous, helping in early diagnosis and medical image analysis.

## Features
- Image classification using CNN
- Binary classification (Cancer / Non-Cancer)
- Model training and evaluation
- Performance analysis using accuracy and confusion matrix
- Can be extended for cancer grading in the future

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Seaborn
- OpenCV
- Scikit-learn
